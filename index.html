Conversão de binário para decimal

A conversão de números da base 2 para base 10 é bastante simples. Basta reescrever o número numa expansão de base 2, conforme o exemplo abaixo.

(0110)2 = 0(3)1(2)1(1)0(0)

0*2³ + 1*2² + 1*2¹ + 0*20 = 0 + 1*4 + 1*2 + 0 = 4 + 2 = (6)10

Sistemas de numeração

Um sistema de numeração é um conjunto de princípios constituindo o artifício lógico de classificação em grupos e subgrupos das unidades que formam os números. A base de um sistema de numeração é uma certa quantidade de unidades que deve constituir uma unidade de ordem imediatamente superior. Os sistemas de numeração tem seu nome derivado da sua base, ou seja, o sistema binário tem base dois, o sistema septimal tem base sete e o decimal tem base dez.

O sistema binário ou de base 2 é um sistema de numeração posicional em que todas as quantidades se representam com base em dois números, ou seja, zero e um (0 e 1). O sistema binário é base para a Álgebra booliana (de George Boole — matemático inglês), que permite fazer operações lógicas e aritméticas usando-se apenas dois dígitos ou dois estados (sim ou não, verdadeiro ou falso, tudo ou nada, ligado ou desligado, 1 ou 0).
